# Dossier médias publics

Placez ici les images versionnées du site.

- Produits: `/media/produits/...`
- Bannières: `/media/bannieres/...`

Ces fichiers sont servis par Vite/Vercel depuis le dossier `public`.

Exemples d'URL valides à saisir dans l'admin:
- `/media/produits/chaussure-01.jpg`
- `/media/bannieres/home-hero.webp`
